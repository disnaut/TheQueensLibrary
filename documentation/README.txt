This folder serves to house any documentation for this project.
It will include things like PDFs of Schema, Developer Help, etc.


Note: The following below are things that will be added to as time goes on.
      If you would like to add to this, please discuss the addition with
      Disnaut first.

Coding Standards:
  - Every variable should be prefaced with a 3 letter identifier for what
    datatype it will be used for.
      3 Letter Identifiers:
        - Sqlite connection -> sql
        - String -> str
        - Boolean -> bln
        - Integer -> int
